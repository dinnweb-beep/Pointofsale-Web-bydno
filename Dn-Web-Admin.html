📤 Export & Sinkron Laporan

📋 Cara membuat Google Sheets Webhook (klik untuk buka)
  1. Buka script.google.com → Buat proyek baru
  2. Buat Google Sheets baru, copy ID dari URL
  3. Paste kode ini ke editor:
function doPost(e){
  var data = JSON.parse(e.postData.contents);
  var ss = SpreadsheetApp.openById("GANTI_ID_SPREADSHEET_ANDA");
  var s = ss.getSheetByName("Laporan") || ss.insertSheet("Laporan");
  if(s.getLastRow()===0) s.appendRow(["Waktu","Total","Detail"]);
  s.appendRow([data.waktu, data.total, data.detail]);
  return ContentService.createTextOutput("OK");
}
  1. Deploy → Web app → Execute as: Me → Access: Anyone
  2. Copy URL → Paste di kolom di atas → klik "Kirim ke Google Sheets"
